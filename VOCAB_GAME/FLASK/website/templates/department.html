    • Digital Reference

    The Digital Reference originates from the Ecsel project Productive4.0, which aims to create a B2B digital platform to connnect End to End semiconductor activities and supply chains containing semiconductors using the Semantic Web and ontologies. The model organizes more than 1000 relevant classes into "lobes", that represent specific aspects serving as knowledge bases for both humans and machines alike. At the application stage the Digital Reference provides resilience by enabling fast actions based on enhanced visibility, enables high knowlege based AI applications and a "truer demand" for the industry and its suppliers, thus enabling a more resilient supply to the customers.
